## What is Tabi?

macOS's built-in `⌘ Tab` switches apps — not windows. If you have six Chrome windows open, you still have to click around after switching.

Tabi replaces that with `⌥ Tab`: a full-screen window picker that shows live thumbnails of every open window across all apps. Press, pick, done.

**No menu bar icon. No preferences window. No bloat.** It runs silently in the background and stays out of your way.

## How it works

| Key | Action |
|-----|--------|
| `⌥ Tab` | Open switcher |
| `⌥ Tab` (hold) | Browse windows |
| Release `⌥` | Switch to selected window |
| `Esc` | Dismiss |

1. Download the DMG below
2. Open it and drag **Tabi** to Applications
3. Launch Tabi — grant Accessibility and Screen Recording when prompted
4. Press `⌥ Tab`

## Permissions

Tabi needs two permissions to work:

- **Accessibility** — to detect the `⌥ Tab` shortcut
- **Screen Recording** — to capture window thumbnails

Both are requested on first launch.
